# HR Recruitment Analysis Dashboard (Excel)

## Project Overview

This project simulates a real-world recruitment analysis scenario in which a People Analytics team investigates recruitment performance across departments, regions, and sourcing channels.

Using Microsoft Excel, the data was cleaned, analysed and transformed into an interactive dashboard to provide insights into hiring efficiency and recruitment outcomes.

---

## Business Problem

Management wanted answers to several key questions:

* Which regions receive the highest number of applications?
* Which recruitment channels generate the most hires?
* How do salary expectations compare with offers?
* Which roles take longest to fill?
* How has recruitment activity changed over time?

---

## Dataset

The dataset contains:

* Applications received
* Candidate information
* Roles applied
* Departments
* Recruitment channels
* Candidate status
* Salary expectations
* Offer salaries
* Hiring dates

Intentional data quality issues included:

* Duplicate records
* Missing values
* Mixed date formats
* Inconsistent source names
* Inconsistent region names

---

## Data Cleaning

The following steps were performed:

* Removed duplicate records.
* Standardised categorical values.
* Cleaned inconsistent text entries.
* Created helper columns for analysis.
* Prepared analysis-ready data.

---

## Dashboard Features

### KPI Cards

* Applications Received
* Interviews Completed
* Candidates Hired
* Average Time to Hire

### Visualisations

#### Regional Recruitment Performance

Compares applications received and hires by region.

#### Recruitment Channel Performance

Shows which channels generate the most applications and hires.

#### Monthly Recruitment Activity Trend

Tracks applications, interviews and hires over time.

#### Average Time to Hire by Role

Identifies which roles take longer to fill.

#### Salary Expectations vs Offers by Role

Compares candidate salary expectations against offered salaries.

### Interactive Filters

* Year
* Gender
* Recruiter
* Department

---

## Key Insights

* Recruitment activity fluctuated throughout the year.
* LinkedIn generated the highest application volume.
* Some roles experienced longer hiring cycles than others.
* Salary expectations differed across roles.
* Regional hiring performance varied significantly.
